# Tic-Tac-Toe AI 🎮

A classic Tic-Tac-Toe game with an unbeatable AI opponent built using Python.

## 🚀 Features
- Play against computer AI using Minimax algorithm
- Clean Tkinter GUI interface  
- Two-player mode also available
- Tracks wins, losses, and draws

## 🛠️ Tech Stack
- **Language:** Python
- **GUI:** Tkinter
- **Algorithm:** Minimax for AI decision making

## 🔧 How to Run
1. Clone the repo: `git clone https://github.com/BunnyxBytes/tic-tac-toe-ai`
2. Run the file: `python Tic-Tac-Toe.py`
